SCHEDULE Batch RUN FOR WINDOWS USER

Compile "cobc -x -j BATCH-UPDATE.cbl -o bin/BATCH-UPDATE.exe
Run "bin\MAIN.exe" and Add user's data
Run schedule batch with ".\run_batch.bat" 
You can change 3 minutes to 5 minutes in "run_batch.bat" folder and change from "/sc minute /mo 3 /f" to "/sc minute /mo 5 /f"
Check updated "Approved/Conditional/Rejected" Result in T_APPLICATION.CSV
You can check next schedule time with "schtasks /query /tn "BatchUpdateTask" /v /fo list"
You can delete back using "schtasks /delete /tn "BatchUpdateTask" /f"
